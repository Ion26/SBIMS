SBIMS — Student Branch Information Management System

A simple, modular Python-based system to manage student information.

SBIMS allows adding, viewing, searching, filtering, updating, deleting, and analyzing student records.
Data is stored in CSV format, and the system follows a clean modular architecture.

FEATURES:
Core Functionalities
Add new students

List all students

Search students by name

Filter by branch

View top N students by GPA

Update student details

Delete a student

ANALYTICS:

Branch-wise student distribution

Average GPA by branch

TECHNICAL FEATURES:

Modular Python code

CSV-based storage

Clear input/output structure

Readable UI menu

Expandable architecture
